# Google AI Agents Intensive Course

This repository contains materials and notebooks for the Google AI Agents Intensive Course.


## Getting Started
1. Clone the repository:
   ```zsh
   git clone https://github.com/harish-anandaramanujam/google-ai-agents-Intensive-course.git
   ```
2. Open the notebook in VS Code or Jupyter Lab.

## Requirements
- Python 3.8+
- Jupyter Notebook or VS Code with Jupyter extension
